💧

التغذية الرياضية
دليل
🥗

تغذية الجسم الرياضي

أكل نظيف + بروتين كافٍ = جسم لياقي

📊 الاحتياج اليومي

  • بروتين: 1.6-2 غ/كغ (بناء وإصلاح)
  • كربوهيدرات: 3-4 غ/كغ (طاقة التمرين)
  • دهون صحية: 0.8-1 غ/كغ (هرمونات ومفاصل)
  • سعرات: صيانة أو عجز خفيف 200 سعرة
  • ماء: 3-4 لترات يومياً

🥩 مصادر البروتين

  • صدور دجاج — 31غ/100غ
  • سمك (تونا، سلمون) — 25غ/100غ
  • بيض — 6غ/بيضة
  • لحم أحمر قليل الدهن — 26غ/100غ
  • زبادي يوناني — 10غ/100غ
  • بقوليات (عدس، حمص)

⏰ التوقيت

  • قبل التمرين (1-2 ساعة): كربوهيدرات + بروتين خفيف
  • بعد التمرين (30 دقيقة): بروتين سريع + كربوهيدرات
  • قبل النوم: بروتين بطيء (زبادي، جبنة)

🚫 تجنّب

  • المشروبات الغازية والعصائر المصنّعة
